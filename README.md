# SVETlANNa
Open source library for simulation and engineering of diffractive neural networks


# Development

## poetry
To init the project and install all dependencies use
```
poetry install
```
Then install PyTorch (the version you want)
```
pip install torch
```

To add package
```
poetry add <name>
```

To add package required only on dev stage use
```
poetry add <name> --group dev
```
